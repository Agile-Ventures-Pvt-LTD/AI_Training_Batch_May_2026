# This prototype is an AI-powered customer support ticket intelligence application that accepts one or more customer tickets and produces a structured analysis package.

# This prototype contains summarization, classification and analysis tasks.

# Also, certain types of prompt- engineering are being implemented, i.e.
- Zero-shot prompting
- One-shot prompting
- Few-shot prompting
- Role-shot prompting

# Zero-shot prompting
- Small size prompt with no example(s).
- Suitable for simple and easy tasks(quesries).
- Not suitable for complex queries.

# One-shot prompting
- One example along with the instruction is given.
- Suitable when output needs to be formatted.

# Few-shot prompting
- Multiple examples with intruction is given.
- Suitable when output needs to be highly formatted.

# Role-shot prompting
- A role is assigned to the LLM model.
- Useful when domain-specific output is needed.
