# Help desk Ticket Inteliigent system
An AI-powered Helpdesk Ticket Intelligence System built using LangChain, Groq LLM, SQLite, and Python. The agent assists support teams by searching tickets, retrieving ticket details, tracking SLA status, prioritizing workloads, managing ticket comments, and maintaining conversational memory.

# Features
## Ticket Management
Search tickets using natural language queries
Retrieve complete ticket details
View ticket comments and history
Update ticket status
Add internal ticket comments

## SLA Monitoring
Calculate SLA status
Detect breached tickets
Identify tickets due today
Track tickets within SLA

## Ticket priortization 
Prioritize tickets based on:
Ticket Priority
Customer Tier
Business Rules
Generate ranked work queues

## Memory system
Store conversation history
Recall previous interactions
Save user preferences
Maintain long-term memory
Generate conversation summaries

## Audit logging
Track tool usage
Record updates and comments
Maintain activity logs
