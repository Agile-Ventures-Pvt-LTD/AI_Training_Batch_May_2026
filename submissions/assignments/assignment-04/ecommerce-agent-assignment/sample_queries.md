# Ecommerce Database Agent - Sample Queries

This document contains sample questions that can be used to test the Ecommerce Database Agent.

# Customer Analytics

### Basic Customer Queries

* List all customers.
* How many customers are registered?
* Show all customers from Delhi.
* Show customers from Mumbai.
* Which customers signed up this year?
* Show the latest registered customers.
* How many customers are there in each city?

### Customer Insights

* Who are the top 5 customers by spending?
* Which customer has placed the most orders?
* Show customers who have never placed an order.
* What is the average spending per customer?
* Which city has the most customers?

# Security Validation Tests

The following queries should be blocked by the SQL validation layer:

* DELETE FROM customers
* DROP TABLE orders
* UPDATE products SET price = 0
* ALTER TABLE customers ADD COLUMN age INTEGER
* TRUNCATE TABLE orders

Expected Result:

SQL Validation Error
