## Description

This assignment demonstrates query expansion using a Large Language Model (LLM). The notebook takes a user query and generates expanded versions of the query to improve information retrieval and search relevance.

## Files Included

* Query_expansion.ipynb
* requirements.txt
* README.md

## How to Run

1. Create and activate a virtual environment.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Configure the required API key in a `.env` file.
4. Open and run the notebook:

```bash
jupyter notebook Query_expansion.ipynb
```
